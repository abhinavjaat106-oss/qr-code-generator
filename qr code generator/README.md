# 📸 My Premium QR Code Generator

Hey there! I built this project because I wanted a simple, clean, and fast way to create QR codes without all the annoying ads and clutter you usually find on those "free" generator sites. 

It's a straightforward tool that does one thing really well: it takes your text or links and turns them into a high-quality QR code instantly.

## 🚀 Why this is cool

- **No more waiting**: It generates the code as you type. Super snappy.
- **Any size you need**: Whether you need a tiny code for a business card or a huge one for a poster, I've got you covered with options from 100px up to 1000px.
- **Easy download**: Just hit the download button and you're good to go. No "right-click and save as" nonsense.
- **Looks great on your phone**: I spent a lot of time making sure it looks just as good on a small screen as it does on a desktop.
- **Clean Design**: I used a nice teal color and the 'Inter' font to keep it looking modern and premium.

## 🛠️ What's under the hood?

I kept it simple with the classic trio:
- **HTML/CSS** for the structure and that sleek look.
- **Vanilla JavaScript** for the logic (no bulky frameworks here).
- **QRCode.js** - a great little library that does the heavy lifting for the QR generation.

## 🏃 How to use it

1. Just grab the files.
2. Open `index.html` in your browser.
3. Type something in, pick a size, and boom—you've got a QR code.
